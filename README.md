# Booster

**Simple Audio Boost Extension**

Booster lets you crank the volume **past 100%** on any website (YouTube, Netflix, Twitch, Spotify, random web videos — everything) using a proper AudioWorklet. No ads, no tracking, super lightweight.

### Features
- Real-time boost with zero quality loss
- Works on every audio/video element (even ones added dynamically)
- Per-tab settings — your boost level stays for that tab only
- Clean popup slider + number input
- Works in all frames (iframe videos included)
- Firefox + Chromium support with one codebase

### How to Install

**Chrome / Edge / Brave**
1. Download `booster-chromium.zip` from the Releases tab
2. Unzip it anywhere
3. Go to `chrome://extensions/`
4. Turn on **Developer mode**
5. Click **Load unpacked** -> select the folder

**Firefox**
1. Download `booster-firefox.xpi`
2. Drag & drop the .xpi file into Firefox (or go to `about:debugging#/runtime/this-firefox` -> "Load Temporary Add-on" for testing)

### How to Use
1. Click the Booster icon in your toolbar
2. Move the slider
3. Done — every sound on the tab is instantly louder
